Android Androwarn Service

Description

The Android Androwarn Service is an AppVet tool service that wraps the androwarn tool to check for various Android vulnerabilities.

Possible Results

  • LOW: Low or no risk detected.

  • MODERATE: Moderate risk assessed.

  • ERROR: An internal problem occurred with the tool.
