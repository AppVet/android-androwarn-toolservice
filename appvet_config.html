AppVet Configuration

  • Step 1: Create AppVet Tool Account
  • Step 2: Install AppVet Tool Adapter
  • Step 3: Test or Install Tool

Step 1. Create AppVet Tool Account

In an AppVet system, online tools are required to have a user account on AppVet in order to log in and post reports and risk assessments for an app. To add an AppVet user account for the Android Service tool, start AppVet (i.e., start Tomcat on the AppVet server) and log in using an ADMIN user account. Next, select Admin > Users and select the Add button. For this tool, set User ID to 'androwarn' and set Role to 'TOOL_PROVIDER' in the Add User dialog box. All other parameters should be set to appropriate values. Remember your selected Password as this will be needed later to configure the Androwarn service.

After adding user 'androwarn', select Submit and shutdown AppVet (i.e., shutdown Tomcat).

Step 2. Install AppVet Tool Adapter

From the AppVet server, open a browser and download the latest Androwarn tool adapter android-androwarn-tool.xml file at:

https://github.com/AppVet/android-androwarn-toolservice/releases


and move it to your $APPVET_FILES_HOME/conf/tool_adapters/android directory.

Next, edit android-androwarn-tool.xml by changing the <appvet:URL> property to reflect the tool server's hostname or IP address. If the Androwarn service resides on the same server as AppVet, then set <appvet:URL> to:

		
	<appvet:URL>http://127.0.0.1:8080/android-androwarn-toolservice/Service</appvet:URL>

If, however, the Androwarn service resides on a different server (e.g., tools.example.com) and uses SSL and port 8443, then set <appvet:URL> to: 

		
	<appvet:URL>https://tools.example.com:8443/android-androwarn-toolservice/Service</appvet:URL>

Note that the '/android-androwarn-toolservice/Service' portion of the URL should never be modified. For more information on Tool Adapter Properties, see Tool Adapter Properties.

Step 3. Test or Install Tool

Test

If you are connecting to an existing Androwarn service that is accessible to AppVet, you can test the service now by logging into AppVet, uploading an app, and viewing the Androwarn report. If you experience errors connecting to the tool, verify the tool adapter properties or contact the tool's host administrator.

Install Tool

If you do not yet have access to an existing Androwarn tool, you must install it. For more information, see the Androwarn Service Installation.
