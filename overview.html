Description

The Androwarn service is an AppVet-compatible online tool that wraps the Androwarn tool to check for various Android vulnerabilities.

Possible Results

  • LOW: Low or no risk detected.

  • MODERATE: Moderate risk assessed.

  • ERROR: An internal problem occurred with this tool.

References

For more information about the Androwarn tool, see Androwarn.
